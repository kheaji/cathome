# 고양이 쇼핑몰 (Cat Home)

고양이 쇼핑몰을 주제로 한 포트폴리오 사이트입니다.

## 프로젝트 개요

- **목적**: 고양이 쇼핑몰을 주제로 한 포트폴리오 사이트 구현
- **주요기능**: 고양이 관련 상품을 온라인으로 구매할 수 있고 커뮤니티 역할도 함
- **타겟사용자**: 고양이용품을 소비하는 적극적인 소비자(Vocal Consumer)

## 기술 스택

### Frontend
- HTML5
- CSS3 (Vanilla CSS)
- JavaScript (ES6+)

### Backend/Database
- Supabase

### 배포
- Vercel

### 버전관리
- GitHub

## 디자인 가이드

### 디자인 컨셉트
모던하고 깔끔하면서도 메인 컬러의 밝은 느낌을 살려서 전달합니다.

### 색상
- **메인 컬러**: #FF6D00
- **보조 메인컬러**: #FFA35F, #FFD8BD
- **포인트 컬러**: #00A8EA
- **보조 포인트컬러**: #6DCDF1, #DAF1F7
- **텍스트 색상**: #111111

### 폰트
- **제목 폰트**: Pretendard Bold
- **본문 폰트**: Pretendard / 18px

## 주요 기능

### 쇼핑몰
- 핫딜
- 신상품
- 랭킹
- 이벤트
- 체험단

### 커뮤니티 (집사 커뮤니티)
- 게시글 작성/조회/수정/삭제
- 이미지 업로드 (1개)
- 댓글 작성/삭제
- 제목 검색
- 로그인한 사용자만 글쓰기
- 작성자만 수정/삭제 가능

## 프로젝트 구조

```
cathome/

│   ├── reset.css       # CSS 리셋
│   └── style.css       # 메인 스타일
```

## 코딩 규칙

### HTML
- 시맨틱 태그 우선 사용 (header, nav, main, article, section, footer)
- 주석은 한국어로 작성
- 들여쓰기는 2칸

### CSS
- 클래스명은 명확한 의미 전달
- Vanilla CSS 사용 (프레임워크 사용 금지)
- 주석으로 섹션 구분

### JavaScript
- ES6+ 문법 사용
- 가독성을 위해 함수는 작게 분리
- 주석은 한국어로 작성

## 반응형 디자인

- **모바일**: 0 ~ 767px
- **태블릿**: 768px ~ 1023px
- **데스크톱**: 1024px 이상

## 라이선스
